# 🛍️yalooStore-shop
API 설계와 문서화 자동 작업을 담당하고 있는 서버입니다.<br>
데이터베이스 모델링 및 설계를 진행하여 JPA를 사용해 RDBMS와 연동하여 데이터를 관리합니다.

## 
## Features
### 회원 관리
- 회원 가입
- 회원 수정
- 회원 탈퇴
  
### 주문
### 상품
- 전체 상품 조회
- 검색, 특정 상품 조회
  
### 위시리스트
- 회원 기반의 위시리스트 추가
- 회원 기반의 위시리스트 수정
- 회원 기반의 위시리스트 삭제
